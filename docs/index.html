How gender was assigned — read this first

Most candidates (≈77%) had no gender recorded in the source data. Gender was predicted automatically using two tools: the gender_guesser library (an open-source name database) and the ONS historical baby names dataset (1904–2024), using birth year where available to account for names whose gender balance has shifted over time (e.g. "Ashley", "Kim").

This approach has significant limitations: it works best for common Western European names and struggles badly with non-Western names — Arabic, South Asian, East Asian and African names are frequently left Uncategorised. Around 16% of candidates remain Uncategorised. Percentages shown throughout this page are calculated only among candidates whose gender was determined using the gender_guesser library or the ONS baby names dataset, so figures for councils or parties with high Uncategorised rates should be treated with caution. The map uses colour saturation to signal this uncertainty — washed-out areas have more Uncategorised candidates.

Gender is treated as binary (male/female) for prediction purposes, except where the original data explicitly records a non-binary identity.
